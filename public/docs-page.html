Permissions and Setup

Dynamic music gets ready with almost zero configuration.

Permissions

@everyone role permissions
  • Add songs to the queue
  • Can able to use commands such as np, queue etc
  • Can use .skip command(it is counted as voteskip)
  • Can use .Bass command(available in premium version)
@DJ role permissions
  • Gets control over all the music commands.
  • Can skip, remove, clear the queue and also disconnect bot from vc
  • Access to all the premium commands
@owner or @admin or @moderator role permissions
  • Those who have manage messages permission gets access to all the commands
  • Same for mods and admins

Restrict commands

To restrict commands to music vc and channel make sure bot doesn't have administrative permission.

  1. Open a channel where you are going to use music commands
  2. Goto Edit channel
  3. Click on Permissions and from members add Dynamic Music
  4. Give access to read and send messages, embed links, external emojis
  5. Follow the same steps with voice channel.

Set DJ Role

Just create a role named "DJ" bot will treat it as dj role. (Make sure rolename shoud be same as this "DJ")

Music

Dynamic Music provides features rich music system which delivers high quality audio with stunning audio effects.

Play Track

Make sure bot has necessary permissions in voice and text channel. Use .play to search and play track. You can check playing song status by using .np The track is played from top youtube search result.

Queue

To check current list of songs use .queue it will show you the list of tracks present in current queue.

Premium

Dynamic Music supports different audio filters. They can be unlocked in premium versions. It is available in four premium tiers. You can check premium pricing at homepage.

Audio effects

  • .bass Applies Bass boost effect. Those who use discord on mobile,they can connect to voice channel by opening discord on web browser. It will deliver stunning and better audio quality.
  • .setpitch You can change pitch of the track
  • .setspeed You can change speed of the track
  • .nightcore You can enjoy nightcore effect by toggling this commands
